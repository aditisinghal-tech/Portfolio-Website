My Projects

Snake Game – Python (Turtle Graphics)

"Old-school fun, coded from scratch!" A classic Snake Game built using Python’s turtle module. Snake grows longer with each food eaten, score updates dynamically with high score saved between sessions, and the game ends on collision with walls or the snake’s own tail. Features simple retro-style graphics and smooth controls.

Tech Stack: Python, Turtle Graphics, File Handling

Password Manager – Python (Tkinter)

A GUI-based password manager built with Tkinter, capable of generating strong random passwords, securely saving them in a .json file, and retrieving them via a search feature. Includes an automatic copy-to-clipboard mechanism using pyperclip and includes robust error handling.

Tech Stack: Python, Tkinter, JSON, Random, Pyperclip, Messagebox

Pomodoro Timer – Python (Tkinter)

A GUI-based time management app to improve focus using the Pomodoro Technique. The app improves concentration, reduces burnout, and boosts productivity.

  • 25-minute work sessions with short and long breaks
  • Visual tomato-themed countdown timer
  • Reset functionality to restart sessions
  • Checkmarks show completed work sessions

Tech Stack: Python, Tkinter, Math, Time Module
